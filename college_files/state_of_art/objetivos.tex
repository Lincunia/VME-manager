\section{Objetivos}
\subsection{Objetivo general}

% Del modelo 3
Crear una aplicación que tendrá 2 finalidades, donde se combinará el
mantenimiento preventivo de hardware y la ciberseguridad para garantizar un
funcionamiento óptimo y seguro de los equipos de cómputo.

\subsection{Objetivos específicos}

\begin{itemize}
  \item
    Desarrollar un sistema de diagnóstico (basado en Ada Test) que permita
    identificar los problemas leves de hardware antes de se generen daños
    críticos.

  \item
    Implementar medidas de ciberseguridad frecuentes
    , como detección de malware, prevención de phishing y protección contra ataques DDoS 
    para la mayor confiabilidad y protección de información de los usuarios.

  \item
    Desarrollar una interfaz fácil de usar para que los usuarios se familiaricen
    para que puedan monitorear y mantener de manera efectiva el estado de sus
    dispositivos.
\end{itemize}
